\begin{abstract}
    This research details the implementation and analysis of a novel, viewpoint-based method for map point removal (MPR) for use in keypoint-based visual SLAM systems. This method uses a K-Nearest Neighbors Observability Model to maintain directional observation statistics for each map point using constant additional space. These statistics allow the existence probability of a point to be continuously estimated via a Bayesian update implemented by an Existence Probability Estimator. This existence probability is then used both to remove outdated map points (e.g. those originally seen on an object that has since moved) and to improve the RANSAC algorithm by prioritizing more reliable map points. We provide the implementation as an open-source library, libVBEE, as well as an ORB-SLAM3 integration utilizing the library for MPR and RANSAC improvement. Additionally, a set of visual-inertial datasets are released, containing scenarios specifically intended to exercise and characterize the performance of the system. % TODO: Discuss effects on well-known, non-dynamic SLAM datasets, along with the new datasets.
\end{abstract}
