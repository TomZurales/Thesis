\begin{abstract}
    This thesis implements and analyzes a novel, viewpoint-aware method for map point culling in keypoint-based visual SLAM systems. This method makes use of a K-Nearest Neighbors Observability Model to estimate viewpoint-dependent observability for each map point using constant additional space per point. This estimate supports continuous calculation of a map point's existence probability using a Bayesian update step implemented by an Existence Probability Estimator. This thesis explores the use of viewpoint-based existence estimations as a method of culling outdated map points, such as those originally seen on an object which has since moved, and as an extension to the RANSAC algorithm to prioritize map points with higher existence probability. The implementation is released as an open-source library, libVBEE, along with an ORB-SLAM3 integration utilizing the library for point culling and RANSAC improvement. Additionally, a set of stereo visual datasets are provided, containing scenarios specifically intended to exercise and characterize the performance of the system. Effects on standard non-dynamic SLAM datasets are discussed alongside the new datasets.
\end{abstract}
