\subsection*{Experiment 1: Observability Model Prediction Quality}
\label{sec:exp1_results}

Table \ref{tab:exp_1_raw} shows the resultant Brier scores for each observability model split by trial. The mean and standard deviation are calculated across trials by model, and are shown in Table \ref{tab:exp_1_aggregated_stats}.

The mean difference between the VBEE observability model Brier scores and each baseline Brier score is shown in Table \ref{tab:exp_1_pairwise_comparison}. Additionally, this table contains a 95\% confidence interval computed using a paired t-distribution over the 10 trials, and a count of the number of trials in which the VBEE observability model ``wins'' against a baseline by achieving a lower Brier score.

The same observability scenarios, training viewpoints and evaluation viewpoints are shared between models in the same trial. Therefore, the Brier scores are considered paired with a trial, giving the following formula for the mean difference between the VBEE observability model and a baseline $b$ as
$$
\overline{\Delta_{vbee,b}} = \frac{1}{10}\sum_{i = 1}^{10} (Brier_{vbee,i} - Brier_{b,i})
$$
where $Brier_{m,i}$ represents the Brier score for model $m$ in trial $i$.
